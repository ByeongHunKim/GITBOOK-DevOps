# Ingress란

{% embed url="https://kubernetes.io/ko/docs/concepts/services-networking/ingress/" %}

* ingress는 클러스터 외부에서 클러스터 내부 서비스로 HTTP와 HTTPS 경로를 노출한다
* 트래픽 라우팅은 ingress 리소스에 정의된 규칙에 의해 컨트롤된다
